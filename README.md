# TradeCalc Hub

Free construction and material calculators for contractors, builders, and DIYers. Static site built with Astro, hosted on Cloudflare Pages.

**Live:** [tradecalchub.com](https://tradecalchub.com)

## Calculators

| Calculator | Category | Description |
|-----------|----------|-------------|
| Concrete | Masonry | Cubic yards, bags (40/60/80 lb), cost for slabs, footings, columns |
| Mulch | Landscaping | Cubic yards and bags for garden beds at any depth |
| Gravel & Stone | Landscaping | Tons and cubic yards by material type and density |
| Paint | Interior | Gallons for walls and ceilings with door/window subtraction |
| Drywall | Interior | Sheets, joint compound, tape, and screws |
| Tile & Flooring | Flooring | Square footage with waste factor for any tile size |
| Fence | Outdoor | Posts, rails, pickets, and concrete by fence type |
| Sod & Grass Seed | Landscaping | Rolls or pounds by grass type and application |
| Brick & Block | Masonry | Bricks, blocks, and mortar for walls and patios |
| Insulation | Interior | Batts, rolls, or blown-in with R-value guidance |

## Tech Stack

- **Framework:** [Astro](https://astro.build/) 6.x (static site generation)
- **Hosting:** Cloudflare Pages (free tier)
- **Analytics:** Cloudflare Web Analytics (free, cookie-free)
- **Contact:** Web3Forms (free tier)
- **Monetization:** Google AdSense + affiliate links (Amazon, Home Depot, Lowe's)

## Development

```bash
npm install        # Install dependencies
npm run dev        # Start dev server at localhost:4321
npm run build      # Build production site to ./dist/
npm run preview    # Preview production build locally
```

Requires Node.js 22.12.0+.

## Project Structure

```
src/
  components/       # Reusable Astro components
  data/             # Calculator definitions, SEO pages, affiliate links
  layouts/          # BaseLayout with header, footer, analytics
  pages/
    calculators/    # 10 calculator pages + category index
    how-much/       # 50+ programmatic SEO pages
    404.astro       # Custom 404 page
    faq.astro       # FAQ with FAQPage schema
    contact.astro   # Contact form (Web3Forms)
    about.astro     # About page
    privacy.astro   # Privacy policy
    terms.astro     # Terms of use
    index.astro     # Homepage calculator grid
  styles/           # Global CSS with design system
public/
  _headers          # Cloudflare security + cache headers
  og-default.png    # Default Open Graph image
```

## SEO Features

- Sitemap auto-generated by `@astrojs/sitemap`
- FAQPage JSON-LD schema on FAQ page
- WebApplication JSON-LD on each calculator
- BreadcrumbList JSON-LD on calculator pages
- HowTo JSON-LD on programmatic SEO pages
- Open Graph + Twitter Card meta tags on all pages
- Canonical URLs on all pages
- 50+ programmatic SEO pages targeting long-tail queries

## Deployment

Connected to GitHub. Every push to `main` auto-deploys to Cloudflare Pages.

Build command: `npm run build`
Output directory: `dist`

## Configuration

After deployment, update these placeholder values:

1. **Cloudflare Analytics token** — `src/layouts/BaseLayout.astro` -> replace `YOUR_CF_ANALYTICS_TOKEN`
2. **Google AdSense** — `src/layouts/BaseLayout.astro` -> uncomment and add publisher ID
3. **Web3Forms key** — `src/pages/contact.astro` + `src/components/FeedbackWidget.astro` -> replace `YOUR_WEB3FORMS_ACCESS_KEY`
4. **Affiliate links** — `src/data/affiliates.ts` -> replace placeholder URLs with actual affiliate links
